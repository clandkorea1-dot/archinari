# 프로젝트명: 의성김씨 아천문중 세보 PWA

## 목표

구글 시트 데이터를 연동하여 가계도 및 문중 정보를 제공하는 앱.

## 기술 스택

- Frontend: HTML5, CSS3 (Tailwind CSS), JavaScript
- Backend: Google Apps Script (API)
- 배포: Netlify

## API 주소

`https://script.google.com/macros/s/AKfycbyy30psVcEtx_A2IY2HqbdyMtFhVeofEr9sFD_pxMU3K9I2n5NuOh6GM3zhwqVoch5D5w/exec`

## 핵심 기능

- 문중원 검색 및 상세 정보 표시
- 8촌 찾기 및 촌수 계산 로직
- D3.js 기반의 동적 가계도 트리 시각화
- 문중 공지사항 및 투표 기능 (Google Forms 연동)
