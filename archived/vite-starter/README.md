# 의성김씨 아천문중 세보 — Vite Starter

이 프로젝트는 Vite + Vanilla JavaScript 기반의 시작용 뼈대입니다.

## 시작 방법

```bash
npm install
npm run dev
```

브라우저에서 개발 서버 주소를 열면 됩니다.

## 포함된 문서
- `docs/final-spec.md`
- `docs/decisions.md`
